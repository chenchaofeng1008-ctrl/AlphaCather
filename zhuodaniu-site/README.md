# AlphaCather

中文名：捉大牛

个人投资内容站，包含文章、视频和投资小工具入口。

## 目录

```text
index.html
articles/
videos/
tools/
assets/
```

## 本地预览

可以直接双击 `index.html` 打开，也可以用任意静态服务器预览。

## Cloudflare Pages 配置

如果 GitHub 仓库只上传本目录内容：

```text
Framework preset: None
Build command: 留空
Build output directory: /
```

如果把本目录放在更大的项目目录里：

```text
Framework preset: None
Build command: 留空
Build output directory: zhuodaniu-site
```

## 当前功能

- 首页：品牌入口
- 文章：占位页
- 视频：占位页
- 工具：工具集合页
- 房屋租金回报测算器：已可用
