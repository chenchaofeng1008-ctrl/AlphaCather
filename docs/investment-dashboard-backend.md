# 投资收益率曲线后端规划

## 当前状态

前端公开页面已完成，当前使用模拟数据展示：

- 今日收益率
- 年初至今收益率
- 累计收益率
- 收益率曲线
- 纳斯达克 / 标普指数对比基准
- 资产类别占比

页面不展示真实金额、具体持仓、交易流水、资金流水和账户标识。

## 后端目标

新增 Cloudflare Worker，提供公开脱敏接口：

```text
GET /api/public/performance
```

接口只返回公开可展示数据：

- 收益率指标
- 收益率曲线点
- 比较基准曲线点
- 资产类别占比

不返回：

- 总资产金额
- 收益金额
- 持仓名称
- 持仓市值
- 交易记录
- 资金流水明细
- 券商账号信息

## 数据同步目标

后续接入 uSMART Open API 后，后端定时同步：

- 交易记录
- 成交记录
- 持仓
- 账户资产
- 资金流水 / 入出金

API 密钥必须保存为 Cloudflare Worker secrets，不进入前端仓库。

## 收益率口径

公开页默认使用时间加权收益率作为主收益率口径，但页面不展示算法术语。

## 下一步

1. 创建 Cloudflare D1 数据库。
2. 执行 `worker/schema.sql`。
3. 部署 `worker/`。
4. 将前端模拟数据替换为 `fetch("/api/public/performance")`。
5. 接入 uSMART Open API。
